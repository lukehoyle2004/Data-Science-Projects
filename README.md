# AI Spam Classification System

## Overview
This project uses a Naive Bayes classifier to detect whether an email is spam or not spam.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Results
- Achieved about 98% accuracy
- Evaluated model performance using classification metrics

## What I Learned
- Text preprocessing
- Tokenization
- Laplace smoothing
- Model evaluation

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the notebook or Python script
